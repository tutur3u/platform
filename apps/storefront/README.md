# Tuturuuu Storefront

Buyer-facing storefront satellite app for Inventory commerce.

- Production host: `https://storefront.tuturuuu.com`
- Local Portless host: `https://storefront.tuturuuu.localhost`
- Fallback API host: `apps/web`

Workspace operators manage catalog, visibility, and Polar settings in Inventory.
This app only renders public/private storefront browsing, carts, and checkout
handoff to Polar.
